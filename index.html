💬 步驟一：建立 LINE 機器人
⚠️ LINE Notify 已於 2025/3/31 停止服務，改用官方 Messaging API（免費）
1
前往 LINE Developers 建立機器人
開啟 developers.line.biz， 用 LINE 帳號登入 → 建立 Provider → 建立 Messaging API 頻道。
2
取得 Channel Access Token
進入頻道 →「Messaging API」分頁 → 找「Channel access token」→ 點「Issue」→ 複製 Token。
📊 步驟二：建立 Google 試算表後台
1
建立 Google Apps Script 專案
前往 script.google.com， 新增專案，將 後台程式碼.gs 全部貼上， 執行 saveToken() 儲存 Token。
2
部署並設定 Webhook
部署為「網頁應用程式」（執行身分：我，存取：所有人）→ 複製網址 → 貼回 LINE Developers 的 Webhook URL → 加機器人好友傳一則訊息取得 User ID。
3
詳細步驟
請參閱資料夾內的 設定說明書.md， 有完整圖文教學，約 15 分鐘完成設定。
部署後複製貼上，以 /exec 結尾
